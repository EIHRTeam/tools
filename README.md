
### 一些神奇妙妙小工具<br>

## 工具列表

### [Image-resizer-tool](Image-resizer-tool/)
AKA 批量图片尺寸转换工具<br>
一个简单易用的批量图片尺寸转换工具，支持自定义输入输出尺寸。<br>
由 [oculto](https://github.com/oculto-link) 编写。

### [skillicon-tool](skillicon-tool/)
技能图标生成工具<br>
这是一个自动化工具，用于将技能图片合成到底图模板上。<br>
由 [ZhuLuOT](https://github.com/ZhuLuOT) 编写。 

### [video-converter](video-converter/)
视频转GIF/WebP转换器<br>
一个视频转GIF/WebP在线转换工具，支持将视频文件转换为自定大小的pdf或WebP文件。<br>
由 [oculto](https://github.com/oculto-link) 编写。
